# RPG Character Creator

A simple and fun browser-based RPG character creator with race and class selection, stat modifiers, and persistent storage.

![Preview] (./assets/rpgCharacterCreator.png)


## Features
- Choose race (Human, Elf, Dwarf, Orc) and class (Warrior, Mage, Rogue, Cleric)
- Automatic stat calculation (HP, Mana, etc.) based on race + class
- Unique ability flavor text
- Add multiple characters to your party
- Delete characters
- Everything saved in browser (localStorage)

## How to Use
1. Just open `index.html` in your browser (no installation needed)
2. Fill in name, pick race & class
3. Click **Create Character**
4. Manage your party with the delete buttons

## Tech Stack
- HTML5
- CSS3 (custom dark fantasy theme)
- Vanilla JavaScript
